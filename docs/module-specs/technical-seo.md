# Technical SEO Module

[PLACEHOLDER — Day 2-3]

This module deploys 5 mu-plugins to a target WordPress site:

1. llms-config.php — site-specific URL config for AI crawlers
2. llms-txt.php — dynamic /llms.txt endpoint
3. ai-crawler-log.php — wp_<prefix>_ai_crawler_log table + tracking
4. markdown-variants.php — ?format=md handler
5. llms-full-txt.php — /llms-full.txt endpoint

Plus:
- ai-crawler dashboard (admin.php?page=ai-crawlers)
