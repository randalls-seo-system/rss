# Article Structure — Longform Guide

## ATF Section (Above The Fold)

```html
<section class="vlnPage">
  <div class="vlnWrap">
    <header class="vlnCard vlnHero">
      <div class="vlnCard-inner">
        <!-- Breadcrumb: Home → Category → Page -->
        <!-- Eyebrow: Topic Label · Subtitle -->
        <!-- H1: Page Title -->
        <!-- Primary Sources: 2-3 .gov/authority links -->
        <!-- Jump Pills: 4-5 anchor links to body H2s -->
        <!-- Hero Lead: 45-55 word intro, answer-first -->
        <!-- CTA Pill -->
      </div>
    </header>
    <!-- 4 Quick Cards with 3-4 fact bullets each -->
    <!-- 3-5 ATF FAQs -->
  </div>
</section>
```

## Main Content Section

```
<div class="vlnPage main-content">

  H2: The Bottom Line Up Front
  - Bold verdict (1-2 sentences)
  - Context paragraph

  H2: [Core Topic Question/Statement]
  - Direct answer (1-2 sentences, ≤30 words)
  - Supporting text (50-70 words total)
  - Structured explanation with rules/thresholds
  - Callout box (Deal Saver, Approval Watchpoint, etc.)

  H2: [Second Major Aspect]
  - Same pattern
  - Comparison table if applicable

  H2: [Third Aspect — Process/Timeline]
  - Same pattern
  - Numbered steps if process-oriented

  H2: [Fourth Aspect — Edge Cases/Gotchas]
  - Same pattern
  - CTA pill mid-article

  H2: The Bottom Line
  - Bold verdict + context paragraph

  H2: Frequently Asked Questions
  - 5-8 FAQs in vlnFaq markup
  - Direct answers, no links

  H2: Resources Used
  - vlnDisclosure block
  - Clean citation anchors

</div>
```

## Word Count Target: 2,000-3,500 words (body only)
## Minimum H2 Sections: 4 (plus BLUF, Bottom Line, FAQ, Resources)
## Minimum Callouts: 3
## Minimum CTA Pills: 2 (distributed, not clustered)
