# Article Structure — Process / How-To

## ATF Section
Same as guide structure. Quick cards should cover prerequisites, timeline, cost, and outcome.

## Main Content

```
<div class="vlnPage main-content">

  H2: The Bottom Line Up Front
  - Bold verdict: what this process involves and how long it takes

  H2: What You Need Before You Start
  - Prerequisites list (bullet-section)
  - Documents, qualifications, timeline expectations

  H2: Step 1 — [First Action]
  - What to do, who to contact, what to expect
  - Timeline for this step
  - Common pitfall callout

  H2: Step 2 — [Second Action]
  - Same depth
  - CTA pill if relevant tool exists

  H2: Step 3 — [Third Action]
  - Same depth

  H2: What to Do If [Common Problem]
  - Troubleshooting the most likely failure point
  - Callout with recovery path

  H2: [Cost / Timeline Summary]
  - Table with steps, timeline, costs

  H2: The Bottom Line
  H2: Frequently Asked Questions
  H2: Resources Used

</div>
```

## Word Count Target: 1,500-2,500 words
## Required: Timeline table or step summary
